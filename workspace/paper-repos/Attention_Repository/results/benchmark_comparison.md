## Task Description

**Synthetic reverse sequence**: given a source sequence of random tokens drawn
uniformly from a 256-token vocabulary, predict the reversed sequence as the target.

This task has a theoretical BLEU ceiling of **100.0** — a correct implementation
that learns the task perfectly will score 100. It is a standard diagnostic for
verifying that encoder, decoder, cross-attention, causal masking, and beam search
all interact correctly.

- Training pairs: 4,000
- Validation pairs: 200
- Test pairs: 200
- Token range: ids 4–255 (ids 0–3 reserved for PAD, BOS, EOS, UNK)
