# Data README

## GeomHerd — Data Generation

This repository does **not** require external dataset downloads.
All data is generated by running the simulator substrates.

---

## Generating CWS Trajectories (Primary Dataset)

The Cividino-Sornette (CWS) substrate is self-contained:

```bash
# Generate full CWS replay set (400 trajectories: 80 seeds × 5 kappa values)
for kappa in 0.5 0.8 1.2 1.8 2.5; do
    python run_detection.py --substrate cws --kappa $kappa --seeds 80 \
        --output_dir data/cws/
done
```

This reproduces the 400-trajectory CWS replay set from Section 3.2.

**Estimated runtime:** ~2-4 hours on CPU (rule-based agents, no LLM).  
With LLM mode (requires `ANTHROPIC_API_KEY`): significantly longer.

## Generating Vicsek Trajectories (OOD Transfer)

```bash
for eta in 0.5 1.0 1.6 2.0 2.5; do
    python run_detection.py --substrate vicsek --eta $eta --seeds 20 \
        --output_dir data/vicsek/
done
```

## LLM Agent Mode

The paper's headline results use **Bedrock Claude Opus 4.6** as the LLM agent
(Section 3.2). To replicate this:

```bash
export ANTHROPIC_API_KEY="your-key-here"
python run_detection.py --substrate cws --kappa 1.8 --seeds 10 --llm_mode true
```

**Note:** LLM persona prompts are intentionally withheld by the authors (Risk R3).
This implementation uses a generic persona template. Results may differ from
the paper's LLM-driven runs; the geometric pipeline is identical.
