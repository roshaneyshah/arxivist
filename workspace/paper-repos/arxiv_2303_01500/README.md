# Dropout Reduces Underfitting (arxiv_2303_01500)

Reproduction of early and late dropout (Liu et al., 2303.01500).

## Contents
- `src/dropout.py` — Early/Late dropout + gradient variance metrics
- `train.py` — smoke-test on tiny ViT
- `configs/config.yaml` — hyperparameters

## Run

python train.py
