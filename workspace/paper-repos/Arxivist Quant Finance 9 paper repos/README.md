# SpotV2Net

**Multivariate Intraday Spot Volatility Forecasting via Vol-of-Vol-Informed Graph Attention Networks**

Alessio Brini (Duke University), Giacomo Toscano (University of Florence)
arXiv: [2401.06249](https://arxiv.org/abs/2401.06249)

*ArXivist-generated reproduction repository. See "Reproducibility Notes" below for
known deviations and assumptions.*

## What this paper does

SpotV2Net forecasts the 30-minute-ahead intraday spot volatility of a cross-section
of assets (the 30 DJIA constituents) using a Graph Attention Network. Each asset is
a graph node; node features are Fourier-estimated spot volatilities and
co-volatilities. Crucially, the edges of the graph carry **volatility-of-volatility**
and **co-volatility-of-volatility** features, which the attention mechanism uses to
weigh spillover effects between assets. The model beats a panel HAR-Spot model, XGBoost,
and an LSTM baseline on both single-step and multi-step (functional) forecasts, and its
predictions can be interpreted with GNNExplainer to reveal which assets drive which
other assets' volatility forecasts.

## Training

```bash
python train.py --config configs/config.yaml
```
Key flags: `--resume <ckpt>`, `--seed <int>`, `--debug` (tiny run), `--dry-run`
(build only, no training).

Set `model.output_dim: 14` and `model.use_edge_features: false` in
`configs/config.yaml` to reproduce, respectively, the paper's multi-step
functional forecast (Sec. 7.4) and the SpotV2Net-NE ablation (Sec. 7.2).

## Evaluation

```bash
python evaluate.py --config configs/config.yaml --checkpoint checkpoints/best.pt --split test
```
Reports aggregate MSE (Eq. 4/7) and QLIKE (Eq. 5/8). Use
`src/spotv2net/evaluation/metrics.py:EvaluationMetrics.diebold_mariano` on the
saved per-observation errors (`results/eval_test.npz`) to reproduce the paper's
Diebold-Mariano significance tables (Tables 3/4/6/7).

## Reproducibility Notes

This repo was generated by ArXivist from the paper's text; a few implementation
details are not explicitly stated in the paper and were assumed (full list in
`sir-registry/arxiv_2401_006249/sir.json` → `implementation_assumptions`):

- **[High risk]** Exact Fourier cutting frequencies (Nc, Mc, Sc, Lc, per-asset
  Nvi/Mvi/Svi/Lvi) are deferred by the paper to an external MATLAB library
  (Sanfelici & Toscano, 2024) and are not given numerically. See
  `data/README_data.md` for calibration guidance.
- **[High risk]** Raw TAQ data is proprietary (WRDS); not redistributed.
- AdamW `beta1`/`beta2`/`weight_decay` and LR schedule are not disclosed —
  PyTorch defaults used, overridable in `configs/config.yaml`.
- Random seed is not disclosed — defaults to 42.
- Multi-head concat-vs-average dimensionality convention: `hidden_dims` in the
  config is treated as each layer's *total* post-aggregation output size
  (see `src/spotv2net/models/spotv2net.py` docstring).
- GNNExplainer's internal optimization hyperparameters are not disclosed — PyG
  defaults used.
