## What This Paper Does

Compares 13 ML methods for forecasting individual stock excess returns using a 920-dimensional predictor set (94 stock characteristics × 9 macro predictors + 74 industry dummies) over a 60-year U.S. equity panel (1957–2016).

**Winner**: Neural networks (NN3: [32, 16, 8] with ReLU + batch norm), tracing gains to **nonlinear predictor interactions** missed by linear methods.

**Best results**:
- Long-short decile portfolio Sharpe ratio: **1.35** (value-weighted), **2.45** (equal-weighted)
- S&P 500 timing Sharpe ratio: **0.77** vs 0.51 buy-and-hold
- Monthly stock-level R²_oos: **+0.40%** (NN3) vs +0.16% (Lewellen OLS-3 benchmark)

## Quick Start

### 1. Install

```bash
git clone <this-repo>
cd asset_pricing_ml
pip install -e .
```

### 2. Run with Synthetic Data (no WRDS needed)

```bash
python train.py --config configs/config.yaml --model NN3 --debug
python train_all.py --config configs/config.yaml --debug
```

### 3. Jupyter Notebook

```bash
pip install jupyter
jupyter notebook notebooks/reproduce_empirical_asset_pricing.ipynb
```

### 4. Run with Real CRSP Data

See `data/README_data.md` for WRDS acquisition. Then:

```bash
# Place data at data/raw/{crsp_returns.parquet, characteristics.parquet, macro_predictors.csv}
python train_all.py --config configs/config.yaml
python evaluate.py --results_dir results/
```
