# Supervised Learning with Quantum-Enhanced Feature Spaces
### Reproduction of Havlicek et al. (2018), arXiv:1804.11326v2

---

## Paper Summary

This repository reproduces two quantum classification protocols:

1. **Quantum Variational Classifier (QVC)** — A variational circuit W(θ) trained by SPSA to find a separating hyperplane in quantum state space.
2. **Quantum Kernel Estimator (QKE)** — A classical SVM using a quantum-computed kernel K(x,z) = |⟨Φ(x)|Φ(z)⟩|².

Both use the feature map 𝒰_Φ(x) = U_Φ(x) H⊗ⁿ U_Φ(x) H⊗ⁿ with:
- φᵢ(x) = xᵢ  
- φ₁₂(x) = (π − x₁)(π − x₂)

for n=2 qubits on data x ∈ (0, 2π]².
